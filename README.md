## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- [Android Studio](https://developer.android.com/studio) (Arctic Fox or newer recommended)
- JDK 11 or higher
- Android SDK with API level 24 or higher
- Git (for cloning the repository)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ashanisuranga2002/Habbit-Tracker.git
   ```

2. **Open the project in Android Studio**
   - Launch Android Studio
   - Select "Open an Existing Project"
   - Navigate to the cloned repository folder and select it

3. **Sync the project**
   - Android Studio will automatically sync the Gradle files
   - Wait for the sync to complete

4. **Build the project**
   - Click on `Build` â†’ `Make Project` or press `Ctrl+F9` (Windows/Linux) / `Cmd+F9` (Mac)

5. **Run the application**
   - Connect an Android device via USB or start an Android emulator
   - Click the "Run" button or press `Shift+F10`
